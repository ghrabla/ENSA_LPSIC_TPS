# Workspace Overview

Short description of each top-level folder:

- tps_code_1: Course/lab exercises covering C, HTML, CSS, JS, and XML topics.
  - tp_c: C language exercises and result text files.
  - tp_html: Basic HTML pages (index, contact, about).
  - tp_css: Intro CSS practice.
  - tp_css_2: CSS layout and pseudo class/element exercises.
  - tp_js: JavaScript exercises (calculator, clock, window management).
  - tp_xml: XML/DTD exercises and validation outputs.

- tps_networking_1: Networking-related resources (Kubuntu Konsole files).
