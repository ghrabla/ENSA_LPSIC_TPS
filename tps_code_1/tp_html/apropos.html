Biographie

Je suis étudiant(e) en Licence Professionnelle d'université en Ingénierie et Conception des Systèmes d'Information (LPU-ICSI) à l'UCA-ENSA SAFI. Passionné(e) par le développement web et les nouvelles technologies, je me spécialise dans la création d'applications web modernes et performantes. Ce projet représente mon apprentissage des fondamentaux du HTML5 et des standards web du W3C.

"Le web est plus une création sociale que technique. Je l'ai conçu pour qu'il ait un effet social - aider les gens à travailler ensemble - et non comme un jouet technique."

- Tim Berners-Lee, inventeur du World Wide Web

Mes compétences

  • HTML5 et sémantique web
  • CSS3 et design responsive
  • JavaScript et programmation orientée objet
  • Bases de données SQL
  • Méthodologies Agile et Scrum
  • Conception UML
  • Git et gestion de versions
